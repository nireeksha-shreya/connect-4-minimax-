# Connect 4 - Minimax AI

A full-stack Connect 4 game powered by a Minimax AI with Alpha-Beta pruning.

## Features
- Human vs AI gameplay
- Minimax algorithm (depth 4)
- Alpha-Beta pruning optimization
- Flask backend
- Interactive frontend (HTML, CSS, JavaScript)

## Tech Stack
- Python
- Flask
- NumPy
- JavaScript
- HTML/CSS

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

## AI Logic
The AI uses:
- Minimax algorithm
- Alpha-Beta pruning
- Board evaluation heuristics
